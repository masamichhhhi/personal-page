# Quickstart: Blog Layout Revamp

## Prerequisites

- Node.js >= 24
- npm

## Setup

```bash
cd /Users/masamichiiimori/masamichhhhi/personal-page
npm install
```

## Run Dev Server

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Format

```bash
npm run format
```
